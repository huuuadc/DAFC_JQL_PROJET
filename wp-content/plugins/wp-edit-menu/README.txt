Edit menu items at bulk. You can edit any menu and delete menu items quickly as desired.

== Description ==

WP Edit Menu is helper plugin which enable dashboard admin to select and remove the selected or all the menu items. It usually comes into play when a menu with large number of menu items are to be edited.

== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `wp-edit-menu.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `wp-edit-menu.zip`
2. Extract the `wp-edit-menu` directory to your computer
3. Upload the `wp-edit-menu` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

== Frequently Asked Questions ==

= How to use? =

Select any menu from dashboard and select the items to remove. You can remove all the items by checking 'check to remove all the items' checkbox. Then, click on the 'Remove Items' button. Now, force reload the page to see the changes.
